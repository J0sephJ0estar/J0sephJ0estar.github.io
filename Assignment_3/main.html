Assignment III: Ray Tracing

Description:

      Here comes the new project. This project may be a little difficult.You have 2 choices:
       1:Based on last project: Let's your sitck man move
       2:Create your own scene, you can place anything in it
      Choose one and use at least two different method such as Phong model, Gouraud model to complete this project.

Design Issues:

      Shape-Drawing Issues: To achieve different shading methods, a new object is drawed to replace the former stickman model here, a sphere. To simulate a view of objects influenced by light, the color of every coordinate pair should be computed separately. So the vertex information is stored in a .vbo(Vertex Buffer Object) file to draw the sphere instead of using methods produced by openGL.
      Shading Issues: A shader is written to be called by the program like a function to compute certain colors. The following are details about shader of three shading methods. They all set a certain light position from which the light comes and use ambient color, diffuse color and specular color to compute the final result but there're differences in interpolation. In float shading, there's no interpolation indeed, and in its shader this is achieved easily by adding a "float" before every "in" and "out" of phong shading or gouraud shading. For phong shading and gouraud shading, the difference lies in their way of interpolating values inside a fragment. The gouraud shading will just using linear interpolation according to its vertex values, but phong shading will carry normal vector interpolation operations according to the normal vector of vertexes before handling one fragment.       Display Issues: To totally complete this assignment, the object simulated should be displayed in a window, and different methods should be put together to compare with each other. So firstly load the .vbo file to draw a sphere, the bind it with these shaders and display it in a window, then use a keyboard listener to switch shading methods by listening to the iuput, binding new shader and redrawing. And that's just all about it.
